# 🧠 SkillMap AI — Skill Gap Analysis

## 🚀 Overview
This project analyzes the gap between **intern skills** and **industry requirements** using machine learning.

## 📊 Features
- Skill extraction from resumes
- Industry demand analysis
- Gap score calculation
- Clustering using K-Means

## ⚙️ Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- KaggleHub

## ▶️ How to Run
pip install -r requirements.txt
python train.py

## 📁 Output
intern_gap_analysis.csv
